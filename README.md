# ATS Dashboard

A fully responsive **Applicant Tracking System (ATS) Dashboard** built with HTML, CSS, JavaScript, and Bootstrap. Features a complete multi-page interface including authentication, candidate management, job tracking, interview scheduling, meetings, reports, and settings.

---

## 🔗 Live Demo
> 🚀 [View Live](https://sakshi-23g.github.io/ats-dashboard)

---

## 🛠 Built With

- **HTML5** — Semantic page structure
- **CSS3** — Custom styling and responsive layouts
- **JavaScript** — Form validation and page interactions
- **Bootstrap 5.3** — Responsive grid and UI components
- **Boxicons** — Icon library

---

## 📁 Project Structure

```
ats-dashboard/
│
├── index.html          # Main dashboard — overview and stats
├── candidate.html      # Candidate listing and management
├── interview.html      # Interview scheduling and tracking
├── job.html            # Job postings management
├── meeting.html        # Meetings overview
├── report.html         # Reports and analytics
├── setting.html        # Application settings
├── login1.html         # Login page with form validation
├── create1.html        # Create account / registration page
│
└── img/                # Images and assets folder
```

---

## ✨ Features

- **Login & Registration** — Form validation with JavaScript
- **Dashboard Overview** — Main stats and summary view
- **Candidate Management** — Track applicants and their status
- **Job Postings** — Manage open positions
- **Interview Tracking** — Schedule and monitor interviews
- **Meetings** — Meeting management interface
- **Reports & Analytics** — Visual reporting section
- **Settings** — Application configuration page
- **Fully Responsive** — Works on desktop, tablet, and mobile
- **Clean UI** — Purple gradient theme with modern card design

---

## 🚀 How to Run Locally

1. Download or clone this repository
```
git clone https://github.com/Sakshi-23G/ats-dashboard.git
```
2. Open the project folder
3. Open `login1.html` in your browser to start from the login page
4. Or open `index.html` to go directly to the dashboard

> No installation, no dependencies, no setup needed.
